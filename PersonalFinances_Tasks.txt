Текущие задачи для Бота (в порядке приоритета)

-- Для версии 1.0

1) Изменить диалог с пользователем в общем чате,
чтобы пользователь мог сам выбрать категорию через интерактивные кнопки

2) После фиксации трат (или в конце рабочего дня) отчитываться обо всех имеющихся активных лимитах

3) Добавление типа лимита

4) Реализация логики обновления переодических лимитов

5) Добавление новой валюты

-- Дальнейшая поддержка

6) Реализация получения экземпляра базы данных через бота (защищенное паролем)

7) Задолжность за предыдущий период (при создании нового лимита)

8) Добавить наименование периода (Чтобы обозначить период как "Ограничения на Ноябрь 2023")

9) Реализация объектов БД через ORM, а не через вызов скриптов напрямую

10) Асинхронность

11) Проверка окончания лимитов и корректное пересоздание повторяющихся

12) Изменение уже внесенных данных

-- Версия 2.0 

12) Формировать корректные отчеты за прошедший период, а так же по запросу в ЛС бота 

13) Предоставить отчеты в текстовом виде или excel/csv/pdf

14) Более простое взаимодействие с ботом в общем чате (Найти способ не тэгать его)

15) Подумать над безопасностью

16) Unit tests

17) Docker + CI
